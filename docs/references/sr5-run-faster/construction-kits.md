# Construction Kits — Character Creation Methods

**Source:** Run Faster, p.62-85
**PDF Pages:** 64-87

---

## Overview

Run Faster provides three alternative character creation methods beyond the standard Priority System: **Sum to Ten** (a flexible variant of Priority), **Point Buy** (800 Karma to build everything from scratch), and **Life Modules** (750 Karma spent through narrative life-stage packages). Each method produces a legal SR5 character but with different levels of flexibility and narrative structure.

---

## Rules

### Sum to Ten

- Players receive **10 points** to distribute across the five priority columns (Metatype, Attributes, Magic/Resonance, Skills, Resources).
- Point costs: A = 4, B = 3, C = 2, D = 1, E = 0.
- A standard Priority character (A, B, C, D, E) also sums to 10, so the budget is identical.
- **Key difference from standard Priority:** each Priority Level (A-E) may be selected **more than once** across different columns, BUT each column (Metatype, Attributes, Magic/Resonance, Skills, Resources) may still only be assigned **one** priority level.
- The E priority costs 0 points and can be taken multiple times, but each column still only gets one selection from the Priority Table.
- Uses either the modified Sum to Ten Priority Table or the standard Priority Table (p. 65, SR5).

> **Cross-reference:** Priority Table values are identical to SR5 core, p. 65. The table on p. 63 of Run Faster reproduces it.

### Point Buy

- Players start with **800 Karma** to build their character entirely from Karma.
- **Step 1 — Buy Metatype:** Purchase metatype from the Metatype Cost Table (p. 64). Set attributes to racial minimums.
- **Step 2 — Buy Everything with Karma:** From this point, use the Character Advancement tables (p. 103, SR5 or p. 106, Run Faster) to buy attributes, skills, qualities, contacts, and gear.
- **Gear conversion:** 1 Karma = 2,000 nuyen. Maximum of **200 Karma** may be converted to nuyen this way (= 400,000 nuyen max).
- **Contacts:** Purchased per the rules on p. 98, SR5.
- **Attribute cap at creation:** Only 1 Mental or 1 Physical attribute may be at its natural maximum. Special attributes (Edge, Magic, Resonance) are exempt from this limit.
- **Magic/Resonance:** To use Magic or Resonance, the character must purchase one of these qualities:
  - **Adept:** 20 Karma. Magic Rating 1. Free power points = Magic Rating. Can buy more Magic ranks with Karma.
  - **Aspected Magician:** 15 Karma. Magic Rating 1. Skilled in one area (Sorcery, Conjuring, or Enchanting). Can buy more Magic ranks.
  - **Magician:** 30 Karma. Magic Rating 1. Full spellcasting, conjuring, and magical abilities. Can buy more Magic ranks.
  - **Mystic Adept:** 35 Karma. Magic Rating 1. Hybrid magician/adept. Does NOT get free power points; must buy power points at 5 Karma per point (max number of power points = Magic Rating).
  - **Technomancer:** 15 Karma. Resonance Rating 1. Can buy more Resonance ranks.
- **Leftover Karma:** Cannot be carried over. Use it or lose it.
- **Leftover Nuyen:** Max 5,000 nuyen carried over from character creation (same as Priority system). Characters roll for starting nuyen per purchased lifestyle using Starting Nuyen Table (p. 95, SR5).

> **Cross-reference:** Character Advancement costs on p. 103 SR5 or p. 106 Run Faster.

> **Ambiguity:** The text says "only advancing a character with the minimum attributes for their metatype and no skills" but does not explicitly state whether the "new skill" or "existing skill" advancement cost applies at creation. The advancement table is the intended reference.

### Life Modules

- Players start with **750 Karma** (lower than Point Buy because modules include built-in bonuses).
- **Step 1 — Buy Metatype:** Same Metatype Cost Table as Point Buy (p. 64). Set attributes to racial minimums.
- **Step 2 — Choose Nationality:** Costs **15 Karma**. Provides primary language, secondary language options, universal skills, and a Region/Demographic within that nationality.
- **Step 3 — Formative Years (age 0-10):** Costs **40 Karma**. Choose one package.
- **Step 4 — Teen Years (age 10-17):** Costs **50 Karma**. Choose one package.
- **Step 5 — Further Education (age 17-19 or 17-21):** Optional. Community College/Trade School = 2 years. Ivy League/State University/Military Academy = 4 years. Cost varies per module (in parentheses in header). If you choose Further Education, you go to Real Life after; if you skip to Real Life, you cannot come back to Further Education.
- **Step 6 — Real Life:** Costs **100 Karma** per module. This is the only category where **more than one module** may be selected, but the same module cannot be taken twice. Represents ~4 years of professional life per module.
- **Step 7 — Finish Karmic Balance:** Spend remaining Karma using advancement tables (p. 107, SR5) for attributes, skills, gear (1 Karma = 2,000 nuyen, max 200 Karma converted), contacts (p. 98, SR5).

#### Life Module General Rules

- **Active skill cap:** No active skill may exceed Rating 7 from modules. If a module would raise a skill above 7, excess ranks are lost.
- **Knowledge skill cap:** Knowledge skills may be raised to a maximum of Rating 9 from modules.
- **Skill group stacking:** If a skill group is selected but previous modules already raised individual skills in that group, add 1 rank to each skill in the group instead.
- **No Karma cost for qualities from modules:** Qualities granted by modules (positive or negative) have no additional Karma cost/bonus. The Karma cost listed next to a quality in a module is its tier/level, not a price the player pays.
- **Magic/Resonance:** Not included in any Life Module. Characters who want magical/resonance abilities must purchase Adept, Aspected Magician, Magician, Mystic Adept, or Technomancer during the finishing step (same costs as Point Buy).
- **Duplicate qualities:** If you receive the same quality twice, select a different quality of the same Karma cost instead. Cannot advance a quality to a higher tier.
- **SIN stacking:** The highest-value SIN received through modules supplants any other SINs gained.
- **Buying off negative qualities:** May buy off negative qualities with leftover Karma. May also pick new positive and negative qualities if under the 25 Karma total limit for negative qualities after all modules are selected.
- **Tours of Duty:** Require a 5-year commitment. Rank 5+ = NCO, Rank 20+ = Officer (see Ranks, p. 86). Can only be taken after completing a Tour of Duty, Law Enforcement, Covert Operations, Shadow Work, or Corporate module.
- **Postgraduate Studies:** Requires completion of a Further Education module.
- **Private Investigator/Detective:** Requires completion of Tour of Duty, Law Enforcement, Covert Operations, Shadow Work, or Corporate module.
- **Preparatory School (Teen Years):** Cannot be chosen if Fugitive or Isolated Rural Upbringing was selected for Formative Years.
- **Magical Education (Teen Years):** Requires purchasing Adept, Aspected Magician, Magician, or Mystic Adept separately (p. 44). After completing, must pick either a Further Education module then Corporate module, or go straight to Corporate module.
- **Military Academy Medical Corps (UCAS/CAS/CFS):** Requires finishing the Nurse or Medicine module from Further Education.

> **Ambiguity:** The book says "skills cannot go above 7 in this process" for Real Life modules but earlier states the cap of 7 applies to all modules. It is unclear if the cap applies only during Real Life or throughout the entire Life Module process. The safest interpretation is that active skills from modules cap at 7 universally.

## Implementation Notes

### Data Model Considerations

- **Sum to Ten** can reuse the existing Priority System infrastructure. The only change is the validation rule: instead of enforcing unique A-E across columns, enforce that point values sum to 10 while each column still gets exactly one level.
- **Point Buy** needs a Karma-spending ledger that tracks all purchases (metatype, attributes, skills, qualities, gear conversion, contacts). The Character Advancement cost formulas must be available at creation time.
- **Life Modules** require an ordered pipeline of module selections. Each module is a bundle of attribute bonuses, skill bonuses, qualities, and languages. The system must:
  1. Track cumulative attribute/skill totals across modules
  2. Enforce skill caps (7 active, 9 knowledge) with excess truncation
  3. Handle skill group stacking (add 1 to each skill if group overlaps)
  4. Manage quality deduplication (swap for same-cost alternative)
  5. Track SIN hierarchy (highest replaces all)
  6. Enforce module prerequisites (e.g., Magical Education -> Further Education or Corporate)
  7. Enforce module exclusions (e.g., Prep School excluded by Fugitive/Isolated Rural)

### Module Data Structure

Each Life Module should be stored as a structured object with:

- `id`: kebab-case identifier
- `name`: display name
- `category`: `nationality` | `formative-years` | `teen-years` | `further-education` | `real-life` | `tour-of-duty`
- `karmaCost`: number
- `attributes`: Record of attribute bonuses (e.g., `{ body: 1, strength: 1 }`)
- `qualities`: Array of quality grants with level (e.g., `[{ name: "Uneducated", level: 8 }]`)
- `skills`: Array of skill bonuses (e.g., `[{ name: "Blades", bonus: 1 }]`)
- `skillGroups`: Array of skill group bonuses
- `knowledgeSkills`: Array of knowledge skill bonuses
- `languages`: Array of language grants
- `prerequisites`: Array of required prior modules/categories
- `exclusions`: Array of incompatible prior modules
- `choices`: Array of player-selected options (e.g., discipline in Further Education, branch in Tour of Duty)

### Priority Table Reuse

The Priority Table for Sum to Ten is identical to the SR5 core Priority Table. If the core table is already in the data layer, Sum to Ten only needs the point-cost mapping and sum-to-10 validation.
