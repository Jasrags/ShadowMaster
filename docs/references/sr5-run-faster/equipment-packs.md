# Pack Your Kit — Equipment Packs

**Source:** Run Faster, p.228-252
**PDF Pages:** 230-254

---

## Overview

Pre-assembled character kits (PACKs) that bundle gear with pre-calculated nuyen and Karma costs. All pack costs are evenly divisible by 2,000, so each PACK can be wholly purchased with the specified amount of Karma at character creation if desired. PACKs frequently note Availability for ease of reference.

---

## General Rules

- Each PACK has a **nuyen cost** and a **Karma cost** (nuyen / 2,000 = Karma).
- Some PACKs also list an **Availability** rating in parentheses (e.g., `(5R)`, `(12F)`).
- Cyber PACKs list **Essence cost** in addition to nuyen/Karma.
- Some Cyber PACKs note **"Requires a Skilljack"** or **"Requires a Skilljack and Skillwires"** as prerequisites.
- Magic PACKs may require **extra Karma** beyond the monetary cost, noted as `+X Karma`.
- The Street Samurai Classic PACK is a **mega-kit** composed of other PACKs (not individual items).
- Ammo crates: ammo type must be defined by weapon category (light pistol, heavy pistol, assault rifle). Crate ammo comes in sets of 10 per category. Ammo is caseless by default; cased ammo may be purchased at no price change.

> **Cross-reference:** Individual item stats, costs, and availability ratings are found in the SR5 Core Rulebook gear tables (p.418-476) and Run & Gun.

> **Ammo Crates Note:** The ammo contained in an ammo crate needs to be defined by the weapon category (i.e., light pistol, heavy pistol, assault rifle) when purchased. ## Implementation Notes

- **Pack ID format:** Use kebab-case derived from pack name, e.g., `intro-runner-pack`, `ares-pistol-pack`, `ammo-crate-1`.
- **Category field:** Use one of: `core`, `weapon-and-ammo`, `armor`, `cyber`, `lifestyle`, `color`, `drone`, `decker`, `vehicle`, `magic`.
- **Cost model:** Store `costNuyen`, `costKarma`, and optionally `costBonusKarma` (for magic focus packs that require extra Karma).
- **Essence:** Store as `essenceCost` (number or null).
- **Prerequisites:** Store as an array of strings, e.g., `["skilljack"]`, `["skilljack", "skillwires"]`.
- **Availability:** Store as string to preserve restriction type, e.g., `"12F"`, `"8R"`, `"4"`.
- **Composite packs:** The Street Samurai Classic Pack should reference other pack IDs rather than duplicating items.
- **Ammo crates:** Items with `*` suffix indicate the ammo type must be specified by the purchaser per weapon category.
- **Item quantities:** Parse quantity from item descriptions (e.g., "Two knives" = quantity 2, "Knife" = quantity 1).
- **Decker Programs issue:** The Basic Cyberdeck Programs Pack table appears to be a production error in the PDF. Verify against errata before implementing.
