# Random Run Generator

**Source:** SR5 Core Rulebook, p. 338–339
**PDF Pages:** 338–339
**Chapter:** Gamemaster Advice — Designing a Run

---

## Overview

The Random Run Tables are a GM utility for quickly generating the skeleton of a run when preparation time is short, inspiration is absent, or a side job needs to drop in mid-session. Rolling against five independent tables produces a complete (if rough) combination of job type, meeting location, target/MacGuffin, employer, and a complicating twist. The results are intended as creative prompts rather than binding constraints — the GM is free to reroll, ignore, or combine results to suit the campaign. The tables work equally well for generating full runs from scratch or injecting a few random elements into an existing plot.

---

## Rules

The GM rolls dice against each table independently. Tables use either 1D6 or 2D6 as specified. All five tables are optional and can be used in any combination.

### Job Type

Roll **1D6**. Determines the broad category of work the runners are being hired to do.

### Meet Locations

Roll **1D6**. Determines where the initial employer meeting takes place.

### MacGuffins

Roll **1D6**. Determines the central object of interest — the target of the run (person, item, data, or location).

### Employers

Roll **2D6** (add results together). Determines who is hiring the runners and for what faction interest. The 2D6 spread creates a probability curve that makes mid-range results (minor corporations, criminal syndicates, megacorps) more common than extreme results (secret societies, exotic beings).

### Twists

Roll **1D6**. Injects a complication or unexpected development into the run. The GM is explicitly not required to keep whatever twist is rolled — it is meant as a creative stimulus.

## Implementation Notes

- The Employer table uses **2D6**, not 1D6. The probability distribution strongly favors results 5–9 (Minor Corp, Megacorp, Criminal Syndicate). Results 2 and 12 are rare.
- The Employer table has two **range rows**: 5–6 (Minor Corporation) and 7–8 (Megacorporation). Store these as range entries, not individual rows.
- All other tables use **1D6** with a uniform distribution — each result has equal probability.
- The MacGuffin result 6 covers three sub-types ("urban building, rural location, or infrastructure object") and could be broken into sub-rolls if more granularity is desired.
- These tables are referenced in the TOC under "Random Run Generator" (p. 478) as an appendix pointer back to this section (pp. 338–339). The standalone appendix page at p. 478 in the printed book is the Master Index; the tables themselves live in the Gamemaster Advice chapter.
- Intended use: roll some or all tables before a session, use results as creative constraints, discard results that do not fit the campaign narrative.
