## Key Concepts

- **Campaign-Centric:** Campaigns define edition, creation method, and gameplay level
- **Transparent Automation:** Automate calculations while surfacing reasoning
- **Reusable Building Blocks:** Factions, scenes, NPCs, and gear catalogs are composable
- **Live Session Flow:** Prep tools feed directly into run mode dashboards
- **Progress Capture:** Every session leaves a durable trail
