# Character Improvement Table

| Improvement                                             | Karma Cost            | Notes                           |
| :------------------------------------------------------ | :-------------------- | :------------------------------ |
| Physical/Mental Attribute                               | New Rating x 5        | Cannot exceed racial max        |
| Special Attribute (Edge/Magic/Resonance)                | New Rating x 5        | Cannot exceed racial max        |
| Active Skill                                            | New Rating x 2        | Cannot exceed linked attribute  |
| Skill Group                                             | New Rating x 5        | All skills must be same rating  |
| Knowledge/Language Skill                                | New Rating x 1        | —                               |
| New Active Skill (Rating 1)                             | 2                     | Must have linked attribute >= 1 |
| New Knowledge/Language Skill                            | 1                     | —                               |
| New specialization (for Active, Knowledge, or Language) | 7                     | Breaks skill groups             |
| New expertise (optional rule)                           | 14                    | Requires specialization first   |
| New Positive quality                                    | Karma cost x 2        | Subject to GM approval          |
| Remove a Negative quality                               | Bonus Karma Value x 2 | Subject to story justification  |
| New spell/ritual/preparation                            | 5                     | Requires appropriate skill      |
| New complex form                                        | 4                     | Technomancers only              |
| New Power Point                                         | 5                     | Cannot exceed Magic rating      |
| New initiate grade (Awakened)                           | 10 + (Grade x 3)      | Grants 1 metamagic              |
| New submersion grade (Technomancer)                     | 10 + (Grade x 3)      | Grants 1 echo                   |
| New Contact                                             | Connection + Loyalty  | Subject to narrative            |
| Improve Contact (Connection or Loyalty)                 | New Rating x 1        | Requires story interaction      |

**Discounts for Initiation/Submersion:**

- Ordeal: −10% (round down)
- Group membership: −10% (round down)
- Both: −20% (round down)
