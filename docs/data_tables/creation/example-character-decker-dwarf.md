# Decker - Dwarf

A dwarf decker with an ethical Code of Honor protecting metahumans. This character excels at Matrix operations with enhanced Logic from a cerebral booster and natural Aptitude for hacking. Diverse technical skills make them useful both in the Matrix and as a field technician.
