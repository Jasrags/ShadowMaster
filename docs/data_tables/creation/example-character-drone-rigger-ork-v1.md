## Notes

1. **Control Rig Integration:** The Control Rig 2 provides +2 to all vehicle/drone limits when jumped in, making this character extremely effective at drone combat.

2. **Hot-Sim Initiative:** With hot-sim enabled (8 + 4D6), this character has excellent Matrix initiative for jumped-in combat, averaging 22 initiative.

3. **Drone Fleet Management:** The RCC (implied) allows running multiple drones simultaneously. The fleet includes:
   - 2× MCT-Nissan Roto-Drone (armed rotary drones)
   - 1× GM-Nissan Doberman (tracked combat drone)
   - 1× Ares Steel Lynx (heavy combat drone with MMG)
   - 2× MCT Fly-Spy (surveillance)
   - 2× Shiawase Kanmushi (micro surveillance)
   - 1× Lockheed Optic-X2 (stealth surveillance)
   - 1× Aztechnology Crawler (utility)
   - 1× Cyberspace Designs Dalmatian (utility/transport)

4. **Vehicle Combat:** The Rover Model 2072 with rigger interface allows jumped-in driving with Ares Alpha assault rifle for mobile fire support.

5. **Juryrigger Quality:** Essential for field repairs on drones and vehicles, providing +2 dice and -1 threshold for temporary fixes.
