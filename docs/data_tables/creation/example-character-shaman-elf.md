# Shaman - Elf

A Salish-Sidhe elf shaman following the path of Bear, specializing in healing and support magic. This character combines traditional tribal shamanism with alchemical knowledge, serving as both healer and spiritual guide. Strong social skills and diverse knowledge make them valuable beyond just magical support.
