# Mercenary / Street Samurai - Human

A highly trained human mercenary with military background, specializing in firearms and heavy weapons. This character follows a strict Code of Honor protecting civilians and non-combatants. An obvious cyberarm with enhanced attributes provides close combat capability, while an extensive arsenal covers every combat scenario from sniper work to room clearing.
