# Alchemist / Investigator - Human

A human mage following the mentor spirit Snake, specializing in alchemy and summoning. This character operates as a private investigator in the magical community, using their arcane talents and preparation-based magic for subtle problem-solving. Strong mental attributes and a vehicle make them mobile and perceptive.
