# Rigger / Wheelman - Troll

A troll rigger specializing in vehicle operation across multiple platforms - ground, water, and air. This character combines a control rig with muscle toners for improved reflexes and maintains a small fleet of armed vehicles. The Middle lifestyle with attached garage provides workspace for vehicle maintenance.
