# Bounty Hunter - Troll

A Salish-Sidhe national troll working as a bounty hunter, likely on parole given the parole officer contact. This character combines traditional hunting skills with modern firearms and extensive gear for pursuing targets. The legal SIN provides legitimate employment options while multiple skill groups make them versatile in the field.
