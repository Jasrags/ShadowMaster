# Assassin / Physical Adept - Elf

A Lakota elf physical adept following the Assassin's Creed, specializing in firearms and infiltration. This character combines supernatural reflexes with extensive weapons training, making them a deadly combatant. High Edge provides luck when it matters most, while adept powers enhance accuracy and reaction time.
