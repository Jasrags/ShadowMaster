# Scatter Diagram & Table

## Scatter Diagram

Roll 2D6 for direction:

- 2, 12: Down
- 3: Down-Left
- 4: Left
- 5: Up-Left
- 6: Slightly Left of Up
- 7: Up
- 8: Slightly Right of Up
- 9: Up-Right
- 10: Right
- 11: Down-Right
