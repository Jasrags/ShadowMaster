# Programs Table

Programs (technically cyberprograms) are files you run on your cyberdeck. While running, a program makes your deck better or gives you more utility. You can't run more than one program of the same type at once. Running programs appear as icons connected to your persona.

| 6      | 18           | 12,000¥ |

### Agent Characteristics

- Uses device's Matrix attributes and agent's rating for skill tests
- Has Computer, Hacking, and Cybercombat skills equal to its rating
- Occupies one program slot
- Has its own persona and icon
- Attacks on an agent damage the device running it (shares Matrix Condition Monitor with deck)
- About as smart as a pilot program of the same rating
