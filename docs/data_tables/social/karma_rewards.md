# Karma Rewards

## Situation

| Situation | Karma |
| :-------- | :---- |
| Character survived | 2 |
| Group complete all objectives | 2 |
| Group completed some objectives | 1 |
| Overall adventure challenge | (Highest opposed Dice Pool / 6 (round down)) |

## Karma Modifiers

| Run Type | Modifier |
| :------- | :------- |
| Standard run | 0 |
| Cold-hearted bastard run | -2 |
| Good feelings run | +2 |
