## Part 2: Resource Spending During Creation

### 2.1 Attribute Points

**Physical Attributes:** Body, Agility, Reaction, Strength
**Mental Attributes:** Willpower, Logic, Intuition, Charisma

**Rules:**

- All attribute points MUST be spent
- Only ONE Physical OR Mental attribute can be at racial maximum
- Special Attributes (Edge, Magic, Resonance) cannot be raised with regular attribute points
- Starting ratings depend on metatype minimums

### 2.2 Special Attribute Points

**Source:** Metatype priority selection (number in parentheses)

**Allocation:**

- Can be spent on: Edge, Magic, or Resonance
- 1 point = +1 rating
- Unspent points are LOST at end of creation
- Edge starts at 1 (2 for humans)
- Magic and Resonance start at 0 (unless set by Magic/Resonance priority)
- Maximum is generally 6 (Edge 7 for humans)

### 2.3 Skill Points

**Individual Skill Points:**

- 1 point = +1 rating to one skill
- 1 point = 1 specialization for a skill

**Skill Group Points:**

- 1 point = +1 rating to entire skill group
- Cannot buy specializations with group points
- Cannot break skill groups in this phase

**Restrictions:**

- Maximum rating 6 at creation (7 with Aptitude quality)
- All skill points must be spent
- Magic/Resonance skills require corresponding attribute

### 2.4 Knowledge & Language Skills

**Free Points:** (Intuition + Logic) × 2

**Rules:**

- 1 point = +1 rating
- Maximum rating 6
- One native language free (rating N)
- Bilingual quality grants second native language

**Knowledge Skill Categories:**
| Category | Linked Attribute |
|:---------|:-----------------|
| Academic | Logic |
| Professional | Logic |
| Interests | Intuition |
| Street | Intuition |

### 2.5 Nuyen (Resources)

**Creation Limits:**

- Availability ≤ 12 (≤10 Street, ≤15 Prime)
- Device Rating ≤ 6 (≤4 Street)
- Augmentation bonus ≤ +4 per attribute
- Carryover limit: 5,000¥ max

### 2.6 Karma-to-Nuyen Conversion

| Level    | Rate             | Maximum Conversion |
| :------- | :--------------- | :----------------- |
| Street   | 1 Karma = 2,000¥ | 5 Karma → 10,000¥  |
| Standard | 1 Karma = 2,000¥ | 10 Karma → 20,000¥ |
| Prime    | 1 Karma = 2,000¥ | 25 Karma → 50,000¥ |

## Part 3: Karma Spending During Creation

### 3.1 Qualities

| Type             | Karma                | Limit           |
| :--------------- | :------------------- | :-------------- |
| Positive Quality | Pay listed cost      | ≤25 Karma total |
| Negative Quality | Receive listed bonus | ≤25 Karma bonus |

### 3.2 Attributes (with Karma)

| Improvement               | Karma Cost     | Notes                       |
| :------------------------ | :------------- | :-------------------------- |
| Physical/Mental Attribute | New Rating × 5 | Still limited to one at max |
| Special Attribute         | New Rating × 5 | No "one at max" restriction |

### 3.3 Skills (with Karma)

| Type                     | Karma Cost     |
| :----------------------- | :------------- |
| Active Skill             | New Rating × 2 |
| Skill Group              | New Rating × 5 |
| Knowledge/Language Skill | New Rating × 1 |
| Specialization           | 7 Karma        |

**Note:** Purchasing a specialization breaks the skill group, preventing future group-based advancement for that skill.

### 3.4 Contacts

**Free Pool:** Charisma × 3 Karma (can only be used for contacts)

| Improvement       | Karma Cost        |
| :---------------- | :---------------- |
| Connection Rating | 1 Karma per point |
| Loyalty Rating    | 1 Karma per point |

**Restrictions:**

- Minimum 2 Karma per contact (at least 1 Connection and 1 Loyalty)
- Maximum 7 Karma per single contact
- No limit on number of contacts

### 3.5 Magic Purchases at Creation

| Item                       | Karma Cost         | Limit                                  |
| :------------------------- | :----------------- | :------------------------------------- |
| Spell/Ritual/Preparation   | 5                  | Magic × 2 total (each category)        |
| Bound Spirit Service       | 1 per service      | # spirits ≤ Charisma                   |
| Focus Bonding              | Force × multiplier | Total Force ≤ Magic × 2; Count ≤ Magic |
| Power Point (Mystic Adept) | 5                  | ≤ Magic rating                         |

**Note:** Adepts receive free Power Points equal to their Magic rating. Mystic Adepts must purchase Power Points with Karma.

### 3.6 Resonance Purchases at Creation

| Item                   | Karma Cost | Limit                |
| :--------------------- | :--------- | :------------------- |
| Complex Form           | 4          | ≤ Logic              |
| Registered Sprite Task | 1 per task | # sprites ≤ Charisma |

### 3.7 Focus Bonding Costs

| Focus Type       | Bonding Cost (Karma) |
| :--------------- | :------------------- |
| Enchanting Focus | Force × 3            |
| Metamagic Focus  | Force × 3            |
| Power Focus      | Force × 6            |
| Qi Focus         | Force × 2            |
| Spell Focus      | Force × 2            |
| Spirit Focus     | Force × 2            |
| Weapon Focus     | Force × 3            |

**Focus Limits at Creation:**

- Maximum bonded foci count = Magic attribute
- Maximum total Force of all bonded foci = Magic × 2
- Only one focus can add Force to any single dice pool test

| Category                  | Limit                      |
| :------------------------ | :------------------------- |
| Karma carryover           | ≤7                         |
| Nuyen carryover           | ≤5,000¥                    |
| Positive qualities        | ≤25 Karma                  |
| Negative qualities        | ≤25 Karma bonus            |
| Gear availability         | ≤12 (standard)             |
| Device rating             | ≤6 (standard)              |
| Attribute augmentation    | +4 max per attribute       |
| Physical/Mental at max    | Only 1                     |

### 4.4 Initiation & Submersion

| Grade | Base Cost | With Ordeal (−10%) | With Group (−10%) | Both (−20%) |
| :---- | :-------- | :----------------- | :---------------- | :---------- |
| 1     | 13        | 12                 | 12                | 10          |
| 2     | 16        | 14                 | 14                | 13          |
| 3     | 19        | 17                 | 17                | 15          |
| 4     | 22        | 20                 | 20                | 18          |
| 5     | 25        | 23                 | 23                | 20          |
| 6     | 28        | 25                 | 25                | 22          |

**Formula:** 10 + (Grade × 3)

**Benefits per Grade:**

- Maximum Magic/Resonance = 6 + Grade
- Learn 1 metamagic (Awakened) or 1 echo (Technomancer)
- First grade grants metaplanar/Resonance Realm access

**Limit:** Grade cannot exceed current Magic/Resonance attribute

### 4.5 Metamagics (Initiation)

| Metamagic          | Effect                                               | Restrictions                      |
| :----------------- | :--------------------------------------------------- | :-------------------------------- |

| Flexible Signature | Alter or fake astral signature                       | Adds grade to Assensing threshold |

### 4.7 Training Time (Optional)

| Improvement                                 | Training Time        | Notes                                     |
| :------------------------------------------ | :------------------- | :---------------------------------------- |
| Attribute (Physical/Mental/Magic/Resonance) | New Rating × 1 week  | May require trainer                       |
| Edge                                        | No time required     | Reflects luck/fortune                     |
| Skill (Rating 1–4)                          | New Rating × 1 day   | May require trainer                       |
| Skill (Rating 5–8)                          | New Rating × 1 week  | May require trainer                       |
| Skill (Rating 9–13)                         | New Rating × 2 weeks | May require trainer                       |
| Skill Group                                 | New Rating × 2 weeks | All skills train together                 |
| Specialization                              | 1 month              | Cannot train anything else simultaneously |

**Trainer Bonus:** 25% time reduction with qualified instructor

### 4.8 Advancement Limits

**Attributes:**

- Cannot exceed racial maximum
- Exceptional Attribute quality allows +1 to one attribute's max
- Augmented maximum = Racial max + 4

**Skills:**

- Active skills cannot exceed linked attribute rating
- Aptitude quality allows rating 7 in one skill (13 post-creation)
- Skill groups require all component skills at same rating

**Restrictions:**

- Cannot awaken after creation
- Cannot emerge as technomancer after creation
- One downtime period: max 2 attribute increases, max 3 skill rating increases

### 6.4 Character Creation Checklist

1. ☐ Choose creation method (Priority, Sum-to-Ten, or Karma Build)
2. ☐ Select metatype (or purchase with Karma if using Karma Build)
3. ☐ Choose magic/resonance path (or mundane)
4. ☐ Select qualities (±25 Karma max each)
5. ☐ Allocate attribute points (all must be spent)
6. ☐ Allocate special attribute points
7. ☐ Allocate skill points (all must be spent)
8. ☐ Allocate knowledge/language points
9. ☐ Purchase gear with nuyen
10. ☐ Spend remaining karma (contacts, skills, attributes, magic)
11. ☐ Verify limits (≤7 Karma, ≤5,000¥ carryover)
12. ☐ Calculate derived statistics
13. ☐ GM approval

### 6.5 Common Validation Errors

| Error                                 | Rule Violated                                    |
| :------------------------------------ | :----------------------------------------------- |
| Two Physical attributes at maximum    | Only one Physical/Mental at max                  |
| Two Mental attributes at maximum      | Only one Physical/Mental at max                  |
| Skill rating exceeds linked attribute | Active skills ≤ linked attribute (post-creation) |
| Gear availability exceeds 12          | Standard creation limit                          |
| More than 7 Karma remaining           | Carryover limit                                  |
| More than 5,000¥ remaining            | Carryover limit                                  |
| Negative karma balance                | Cannot go below 0                                |
| Spells exceed Magic × 2 limit         | Creation limit per category                      |
| Complex forms exceed Logic            | Creation limit                                   |
| Bonded foci exceed Magic × 2 Force    | Creation limit                                   |
| Attribute points not fully spent      | All must be allocated                            |
| Skill points not fully spent          | All must be allocated                            |
