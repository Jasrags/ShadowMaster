## Setup Options

### Automated Setup (Recommended)

Use the setup scripts for automatic installation:

- **macOS/Linux**: `./scripts/setup.sh`
- **Windows**: `.\scripts\setup.ps1`

### Manual Setup

Follow the [Setup Guide](SETUP.md) for step-by-step manual installation instructions for your platform.

## Platform Support

ShadowMaster supports:
- ✅ **macOS** - Native support with Makefile
- ✅ **Linux** - Native support with Makefile
- ✅ **Windows** - PowerShell scripts or WSL (Windows Subsystem for Linux)
