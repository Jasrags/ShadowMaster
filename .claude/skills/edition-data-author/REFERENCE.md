# Edition Data Quick Reference

## Common Availability Values (SR5)

| Availability | Meaning                               |
| ------------ | ------------------------------------- |
| 0-4          | Common, easily obtained               |
| 5-8          | Uncommon, requires some effort        |
| 9-12         | Rare, maximum at character creation   |
| 13-18        | Very rare, require contacts/legwork   |
| 19+          | Extremely rare, military/experimental |

Legality suffixes:

- (none) = Legal
- `"restricted"` = R suffix in book, requires license
- `"forbidden"` = F suffix in book, illegal

## Firing Modes

- `SS` = Single-Shot
- `SA` = Semi-Automatic
- `BF` = Burst Fire (3 rounds)
- `FA` = Full Auto (6+ rounds)

Combined: `SA/BF`, `SA/BF/FA`, `BF/FA`

## Damage Codes

Format: `{damage}P` or `{damage}S`

- `P` = Physical damage
- `S` = Stun damage

Modifiers:

- `(f)` = Fire damage (e.g., `5P(f)`)
- `(e)` = Electrical damage
- `STR` = Use character Strength (e.g., `(STR+2)P`)

## Drain Formulas (Spells)

Format: `F+X` or `F-X` where F = Force

Common values:

- `F-4` = Very easy drain
- `F-3` = Easy drain
- `F-2` = Standard drain
- `F-1` = Moderate drain
- `F` = Standard
- `F+1` = Difficult
- `F+2` = Area/powerful versions

## Fading Formulas (Complex Forms)

Format: `L+X` or `L-X` where L = Level

Common values:

- `L-2` = Easy fading
- `L-1` = Standard fading
- `L` = Moderate
- `L+1` = Difficult
- `L+2` = Complex

## Adept Power Cost Types

- `fixed` - One-time cost (e.g., Killing Hands = 0.5 PP)
- `perLevel` - Cost × level (e.g., Combat Sense = 0.5 PP/level)
- `table` - Custom costs per level (e.g., Improved Reflexes)

## Spell Types

- `mana` - Affects living/astral, blocked by Counterspelling, no effect on tech
- `physical` - Affects physical world, works on cameras/sensors

## Spell Ranges

- `self` - Caster only
- `touch` - Physical contact required
- `LOS` - Line of sight, single target
- `LOS(A)` - Line of sight, area effect

## Spell Durations

- `instant` - Immediate effect, cannot be dispelled
- `sustained` - Active concentration required (-2 to other actions)
- `permanent` - Lasts until healed/dispelled
